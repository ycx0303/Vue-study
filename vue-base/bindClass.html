使用v-bind操作元素的class列表和内联样式

传给v-bind:class一个对象，来动态切换class：

v-bind:class="{ 'bg-red': isRed }"

上面的语法表示 bg-red 这个 class 存在与否将取决于数据属性 isRed 的 truthiness。 你可以在对象中传入更多属性来动态切换多个 class。

绑定返回对象的计算属性

v-bind:class="classObject" 绑定的是对象的计算属性

            computed: {
                classObject: function () {
                  return {
                    'bg-yellow': this.isActive && !this.error,
                    'text-danger': this.error && this.error
                  }
                }
              }

绑定内联样式

v-bind:style="styleObj" 直接绑定到样式对象，让模板更清晰

            data: {
                styleObj: {
                    width: '600px',
                    height: '300px',
                    background: '#eee',
                    borderRadius: '10%'
                }
            }
